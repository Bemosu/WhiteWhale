## 3. 개발 환경

- 개발 인원 : 1인
- 개발 기간 : 2024.06 ~
- 사용기술

1. 메인 : React, TypeScript
2. 빌드 툴 : Vite
3. 스타일 : Tailwind CSS
4. 서버 상태 관리 : @tanstack/react-query
5. 클라이언트 상태 관리 : React Context API
6. Back-End : Firebase(FireStore, Firecloud)
7. 배포 : Vercel
8. 페이지 라우팅 : React Router

7. 장바구니 페이지
8. 물건등록 페이지
9. 물건구매페이지
